# Figma AI Prompt – OptiVerse OS

Design a premium enterprise SaaS web application called **OptiVerse OS – Autonomous System Optimization Platform**.

The interface should feel like a combination of **Linear, Vercel Dashboard, Datadog, Grafana Cloud, Stripe Dashboard, Figma, and Apple Human Interface Design**, but with its own futuristic identity. Avoid generic admin dashboard templates.

---

## Design Style

Create a modern AI-native operating system interface.

Keywords:

* Premium
* Enterprise
* AI-first
* Clean
* Futuristic
* Glassmorphism (subtle only)
* Minimalistic
* Professional
* Dark-first interface
* Data-rich
* Scientific
* System engineering aesthetic
* High-end startup product

The interface should look like software used by Google DeepMind, NVIDIA, Microsoft Azure, OpenAI, or SpaceX engineers.

Avoid bright gradients everywhere.

Avoid childish icons.

Avoid card overload.

Everything should feel precise and engineered.
